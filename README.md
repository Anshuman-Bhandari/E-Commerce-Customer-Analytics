# E-Commerce Customer Experience Analytics

## Overview

Developed a Power BI analytics solution to analyze customer behavior, conversion funnels, revenue trends, and customer lifetime value (CLV) for an e-commerce business. The project focuses on transforming raw data into actionable business insights through data modeling, KPI development, and interactive dashboards.

## Objectives

* Analyze customer acquisition, engagement, and retention.
* Identify conversion bottlenecks and cart abandonment trends.
* Monitor revenue performance and customer lifetime value.
* Enable data-driven business decision-making through dashboards.

## Key Features

* Star Schema Data Modeling (Fact & Dimension Tables)
* Customer Segmentation and CLV Analysis
* Conversion Funnel Analysis
* Revenue and Sales Performance Tracking
* KPI Dashboard Development
* Interactive Power BI Visualizations

## Technologies Used

* Power BI
* DAX
* SQL
* Data Modeling
* Star Schema
* ETL/Data Transformation
* Exploratory Data Analysis (EDA)

## Key KPIs

* Conversion Rate
* Cart Abandonment Rate
* Customer Lifetime Value (CLV)
* Average Order Value (AOV)
* Revenue Growth
* Customer Retention Rate
* Bounce Rate
